* gstalk
** prerequisites
- both transmitter & receiver need pipewire
- ensure direct UDP connectivity (because STUN/TURN is a fuck™, just use tailscale or smth)

** run
=gstalk <rx port> <tx host> <tx port>=
where =rx port= is a UDP port on which you wanna receive audio,
=tx host= and =tx port= is the peer you wanna send audio to

then, pick nodes you wanna record using arrow up/down
and toggle recording with space or enter.
enable autoadd by pressing a, this will automatically
record new audio sources as they pop up.
deafen & mute yourself by respectively pressing d and m.

that's it :3
