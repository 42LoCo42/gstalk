* gstalk
** prerequisites
- both transmitter & receiver need pulseaudio or pipewire-pulse
- ensure direct UDP connectivity (because STUN/TURN is a fuck™, just use tailscale or smth)

** receiver
- run =gstalk rx <port>= to listen on that UDP port

** transmitter
- run =gstalk ls= to list all pulseaudio sink inputs (aka apps that play audio)
- grab the device ID of the thing you wanna stream
- run =gstalk tx <ID> <receiver host> <receiver port>=
